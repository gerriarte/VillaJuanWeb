# Villa Juan - Ecogranja Tenjo

Una experiencia inmersiva en la naturaleza de Tenjo, Cundinamarca. Sitio web desarrollado con Next.js, Tailwind CSS y Framer Motion.

## 🌿 Sobre Villa Juan

Villa Juan es una ecogranja especializada en:
- **Coaching Equino**: Metodología vivencial con caballos para desarrollo personal y liderazgo
- **Eventos**: Corporativos y sociales en espacios naturales únicos
- **Restaurante**: Gastronomía al barril con técnicas artesanales
- **Salidas Pedagógicas**: Educación ambiental y experiencias de aprendizaje

## 🚀 Tecnologías

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS** con sistema de diseño "Organic-Full"
- **Framer Motion** para animaciones fluidas
- **Vercel** para deployment

## 🎨 Características del Diseño

### Atmósfera Dinámica
El sitio implementa transiciones de color de fondo basadas en scroll que cambian según la sección del Coach Equino:
- **Verde** (Trote): Bosque, serenidad, crecimiento
- **Rojo** (Trocha): Arcilla, fuego del barril, energía  
- **Amarillo** (Paso Fino): Atardecer, trigo, exclusividad

### Paleta de Colores
- **Base**: #F9F4F0 (Hueso/Arena suave)
- **Texto**: #2D2926 (Carboncillo orgánico)
- **Acentos**: #634832 (Nogal)

### Tipografía
- **Títulos**: IvyPresto Display / Recoleta (serif)
- **Subtítulos**: Montserrat (sans-serif)
- **Cuerpo**: Inter / Open Sans (sans-serif)

## 📱 Funcionalidades

### Integración WhatsApp
Todos los CTAs generan mensajes contextuales únicos:
- Eventos corporativos y sociales
- Reservas de restaurante
- Salidas pedagógicas
- Planes de coaching específicos

### Secciones Principales
1. **Hero**: Hook emocional y navegación a servicios
2. **Eventos**: Corporativos y celebraciones sociales
3. **Restaurante**: Proceso al barril y menú digital
4. **Salidas Pedagógicas**: Educación experiencial
5. **Coach Equino**: Tres planes con atmósferas dinámicas
6. **Blog**: "Crónicas de la Tierra" con 15 artículos
7. **Contacto**: Ubicación, mapa y información

\`\`\`

## 📦 Deployment en Vercel

El proyecto está optimizado para deployment automático en Vercel:

1. Conectar repositorio a Vercel
2. Las configuraciones están en `vercel.json`
3. Variables de entorno (si las hay) en dashboard de Vercel
4. Deploy automático en cada push a main
